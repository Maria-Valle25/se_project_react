# Project 10: WTWR

## Project Description

WTWR is a responsive weather-based clothing recommendation app.
It fetches real-time weather data and suggests appropriate clothing items based on the current temperature and conditions.

## Tech Stack

- HTML
- CSS
- Responsive Design
- Flexbox
- Grid
- Media Queries
- BEM
- Git & GitHub
- Figma
- React
- JavaScript (ES6)
- API's
